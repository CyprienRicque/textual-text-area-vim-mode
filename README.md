# Textual Text Area - Vim mode

Vim mode for textual Area.

## Normal mode

### Motions
- `h` - move left
- `j` - move down
- `k` - move up
- `l` - move right
- `w` - move to next word start
- `b` - move to previous word start
- `$` - move to end of line
- `0` - move to column 0 (beginning of line)
- `_` - move to first non-blank character

### Insert mode entry
- `i` - insert at cursor
- `a` - insert after cursor
- `I` - insert at line start
- `A` - insert at line end

### Other
- `x` - delete character under cursor
- `v` - enter visual mode
- `V` - enter visual line mode

## Visual line mode

This is best effort because of Textual's limitation: The cursor is always at the end of the selection.
